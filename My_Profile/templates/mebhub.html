MEB Hub

Overview

MEB Hub is a web application created to help university students navigate campus life more efficiently. It provides access to campus maps, upcoming events, and transport schedules across multiple university campuses. The platform aims to centralize key student services into one accessible and mobile-friendly solution.

Technologies Used

Python Django JavaScript HTML CSS PostgreSQL Figma

Key Features

  • Interactive campus map for locating facilities
  • Event listing with date, time, and descriptions
  • Transport schedule viewer with bus route info
  • Mobile-responsive design for easy access
  • User-friendly layout tailored for student needs

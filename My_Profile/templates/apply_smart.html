Apply Smart

Overview

Apply Smart is a web application designed to assist South African high school students in calculating their Admission Point Score (APS) and matching it with university programs. It allows students to input their subject marks and returns a list of qualifying courses based on university-specific requirements. The tool simplifies the research process for students navigating tertiary education options.

Technologies Used

Python Flask WTForms JavaScript HTML CSS

Key Features

  • Calculate APS based on subject marks
  • Match APS with available university courses
  • Dynamic form using WTForms for subject inputs
  • Mobile-responsive UI for students on the go
  • Real-time feedback on qualifying results
